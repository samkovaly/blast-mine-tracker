# Blast Mine Tracker
This plugin lets you track all XP and Profit related statistics of your Blast Mine Runs while you blast.
An important feature of this plugin is that it shows how much XP you are to gain by collecting your ores, which helps an efficient blaster time their collections so that they level up with them, and levels determine XP and profit rates.

## Settings
Only one setting which shows the ore distribution overlay in the top left, but this information is duplicate from the traker information.
